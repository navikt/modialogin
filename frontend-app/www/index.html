Modialogin frontend-image from root

Om denne dukker opp så har du forhåpentligvis startet appen i test-modus. Hvis ikke kan det tyde på at docker-imaget ikke er riktig satt opp.

Eksempel på Dockerfile: 

FROM docker.pkg.github.com/navikt/modialogin/frontend:ecd23b296051ce8d436025e242b79521d9244b32
# Hvor "build" mappen inneholder filene du ønsker skal hostes.
COPY build /www
Denne teksten er her for å verifisere at innlasting av CSS fra en nøstet mappe fungerer. Status: Er denne grønn er alt ok.

Environment variables $env{APP_NAME}: $env{APP_NAME}

Environment variables $env{APP_DUMMY_SECRET}: $env{APP_DUMMY_SECRET}
